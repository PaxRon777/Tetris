# Tetris Clone

Tetris clone built with vanilla JavaScript and HTML5 Canvas. Featuring a sleek dark aesthetic, smooth animations.

## 🚀 Features
-  Charcoal and neon color palette.
- **Responsive Design**: Full-screen layout that adapts to various screen sizes.
- **Next & Hold Systems**: Preview upcoming pieces and hold current ones for strategic play.
- **Ghost Piece**: Visual indicator of where the next piece will land.- 
- **Dynamic Difficulty**: Leveling system that increases drop speed as you clear lines.

## 🎮 Controls

### Desktop
| Action | Key |
| :--- | :--- |
| Rotate Left | `Z` |
| Rotate Right | `X` |
| Move Left | `←` |
| Move Right | `→` |
| Soft Drop | `↓` |
| Hard Drop | `Space` |
| Hold Piece | `C` |
| Pause / Resume | `P` |

## 🛠️ Installation & Setup
Since this is a single-file web project, no installation is required:
1. Clone this repository or download the files.
2. Open `tetris.html` in any modern web browser.
